Web Browser Support

SheerID officially supports the following browsers for accessing our hosted verification applications, including user interface components provided by the JavaScript API. While we support a large number of versions, users are encouraged to upgrade to one of the 2 most recent stable releases of any browser listed below in order to achieve the highest level of performance.

Browser Supported Versions
Google Chrome 18+
Mozilla Firefox 15+
Internet Explorer 8+
Safari 5+
Opera 11+
