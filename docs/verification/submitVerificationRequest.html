Submit Verification Request

Submit a prepared verification request. Request must be in the NEW status as created by Prepare Verification Request.

Request Details

HTTP Method(s)

POST

Resource URI

https://services-sandbox.sheerid.com/rest/0.5/verification/{requestId}/submit https://services.sheerid.com/rest/0.5/verification/{requestId}/submit

Path Parameters

Name Type Description
requestId string the identifier returned with the original request

Response Details

Expected Response

A single VerificationResponse object

application/json
