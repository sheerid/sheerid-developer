Search

Obtain a list of requests that match either name or email details.

Request Details

HTTP Method(s)

GET

Resource URI

https://services-sandbox.sheerid.com/rest/0.5/verification/search https://services.sheerid.com/rest/0.5/verification/search

Parameters

Name Type Description
metadata key (any string prefixed with :) string metadata value for the supplied key (ex: :orderId=179365)
BIRTH_DATE string Complete birth date of the verification subject
EMAIL string An email address, preferably the one issued by the organization being verified
FIRST_NAME string Given name of the verification subject
LAST_NAME string Surname/family name of the verification subject
STATUS_START_DATE string The date on which the affiliation being verified began. For VETERAN, this is the date of discharge/end of active duty status.

Response Details

Expected Response

A list of VerificationResponse objects

Response Content Type(s)

application/json

Errors (HTTP Status Codes)

HTTP Status Code Description
400 if an invalid combination of fields is specified
