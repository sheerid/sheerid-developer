Report Data Descriptions - Extended Report Format

The following information is included in the extended verification reports available to privileged users through the API and reporting UI.

Field Name Description
Request ID A unique identifier for the request
Request Timestamp The date and time of the original request or its most recent revision
Account ID An identifier for the account that submitted the request
Person Id A unique string which represents an individual's identifying data, can be used to recognize duplicate submissions with the same data
User ID An identifier for the user that submitted the request
Username The username of the user that submitted the request
Result
  • true if the individual has been matched in at least one Verification Source and currently has at least one of the requested affiliation types
  • false if the individual has been matched in at least one Verification Source and does not currently have at least one of the requested affiliation types
  • [empty] if the individual has not been matched in any Verification Source
Status
  • NEW if the request has not been submitted to any Verification Source
  • PENDING if the request is still awaiting a final response from one or more Verification Source
  • OPEN if the request has been submitted to one or more Verification Sources but the individual has not been matched
  • COMPLETE if the request has been submitted to one or more Verification Sources and the individual has been matched in at least one
Errors A list of errors encountered during the verification process for this request
OrganizationEntity ID The identifier for the requested organization
Organization Name The name of the requested organization
Organization State The State (location) of the requested organization
Organization Country The Country (location) of the requested organization
OrganizationEntity Type The type of organization represented by the request
Verification Types The type(s) of verification used to obtain a verification result
Requested Affiliation Types The types of affiliations that the request was attempting to verify
Confirmed Affiliation Types The types of affiliations that the individual was determined to have
Data Updated Date The date the affiliation information was most recently updated by the data source
Period Begin Date The start of the most recent affiliation status period
Period End Date The end of the most recent affiliation status period
various Zero or more fields which represent the information of the individual being verified. The presence and number of fields will vary depending on the requested report and the target account's data retention policies.
various Zero or more fields which represent additional information collected with the verification request. The presence and number of fields will vary depending on the requested report and the target account's configuration.
Response Data
Record Type The record type being represented by this report data entry
Error Codes
Verified? true if the individual had a current affiliation which matched one of the requested types, false otherwise
Response Time The time elapsed between the original or modified request and the slowest response from a verification source
Response Count The number of times a verification source was consulted for this request
Duplicate true if this request was determined to be a duplicate submission of a previous request, and as such will not result in a verification fee, false otherwise
Sheer Id Test true if this request was determined to be a test submitted by a sheerid employee, and as such will not result in a verification fee, false otherwise
Revisions The number of times this verification request was modified with updated information and resubmitted
Series RequestID The identifier of the first verification request in a grouped series of verification requests per our duplicate matching logic
Series Index The index of this request within the duplicate verification request series (0 represents the original request in the series)
Request Time Offset The time (in milliseconds) between this request and the original request in the series
Document Reviews The number of document reviews that were performed for this request
Verification Sources Of Record A list of verification sources that were able to locate the individual and conclusively answer the verification request
Verification Types Of Record A list of verification types that were able to locate the individual and conclusively answer the verification request
All Verification Sources Consulted A complete list of verification sources that were consulted to obtain a result for this verification request, regardless of result
Affiliation Attributes A list of extended attributes that further qualify the confirmed affiliations
Cache Usage The type of caching used to satisfy this request
Cached Index The number of cached verifications that have preceded this one based on the same original response
Debits A comma-delimited list of charges associated with this activity, if applicable
Credits A comma-delimited list of credits associated with this activity, if applicable
Campaign Name The name of the verification campaign (template) used for this verification request
Bulk Batch Id Identifier for the bulk batch (if this request was processed via bulk verification)
Additional Data Services List of additional named data services that were used in processing this request
