Count Tasks By Organization Type

Get the counts of available tasks, grouped by Organization Type

Request Details

HTTP Method(s)

GET

Resource URI

https://services-sandbox.sheerid.com/rest/0.5/asset/review/queues/orgtype https://services.sheerid.com/rest/0.5/asset/review/queues/orgtype

Parameters

Name Type Description
status TaskStatus (default value: PENDING)
Possible values: COMPLETED, ESCALATED, IN_PROGRESS, ON_HOLD, PENDING

Response Details

Expected Response

A map keyed on review priority with values equal to the current number of active tasks in each

Response Content Type(s)

application/json
