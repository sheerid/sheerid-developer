Upload

Upload an asset.

Request Details

HTTP Method(s)

POST

Resource URI

https://services-sandbox.sheerid.com/rest/0.5/asset https://services.sheerid.com/rest/0.5/asset

Accepted Content Type(s)

multipart/form-data

Parameters

Name Type Description
assetType AssetType the type of document the user is uploading
Possible values: ACCEPTANCE_LETTER, ARTICLES_OF_INCORPORATION, BUSINESS_TAX_FORM_PAGE1, CLASS_SCHEDULE, DATED_ID_CARD, DBA_REGISTRATION, DD214, ID_CARD, INSURANCE_CARD, IRS_SCHEDULE_B1_FORM_1065, IRS_SCHEDULE_C_OR_CEZ_PAGE_1, IRS_SCHEDULE_G_FORM_1120, IRS_SCHEDULE_K1_FORM_1065, IRS_SCHEDULE_K1_FORM_1120S, LEASE_AGREEMENT, LETTER_OF_AUTHORIZATION, OFFICIAL_LETTER, OTHER, PAY_STUB, POWER_OF_ATTORNEY, REGISTRATION_RECEIPT, REPORT_CARD, SECRETARY_OF_STATE_FILING, TRANSCRIPT, TUITION_RECEIPT, VENDOR_STATEMENT
assetTypeDescription string free form text to provide additional details of assetType
failure string A URL to which the request will be redirected upon a failed upload. If omitted, will not redirect upon failure.
file file one or more files, with parameter keys prefixed by file (Ex: file0, file1...). File type must be one of our supported MIME types.
mergeMultipleDocuments boolean to merge multiple documents into one when possible (default value: true)
success string A URL to which the request will be redirected upon a successful upload. If omitted, will not redirect upon success.
token string a valid asset token obtained from Issue Token

Response Details

Expected Response

A list of Asset objects

Response Content Type(s)

application/json

Errors (HTTP Status Codes)

HTTP Status Code Description
400 either no files have been supplied or at least one file is larger than the maximum upload size
401 the asset token supplied is invalid, expired or has already been used to perform an upload
403 if the request state does not allow upload
415 at least one file is an unsupported MIME type
