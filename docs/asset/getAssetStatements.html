Get Asset Statements

Generate field-level assertions used to perform review of documents, based on the type of asset chosen.

Request Details

HTTP Method(s)

GET

Resource URI

https://services-sandbox.sheerid.com/rest/0.5/asset/{assetId}/review/statement https://services.sheerid.com/rest/0.5/asset/{assetId}/review/statement

Path Parameters

Name Type Description
assetId string the id of an asset that will be reviewed

Parameters

Name Type Description
assetType AssetType the type of asset that has been uploaded
Possible values: ACCEPTANCE_LETTER, ARTICLES_OF_INCORPORATION, BUSINESS_TAX_FORM_PAGE1, CLASS_SCHEDULE, DATED_ID_CARD, DBA_REGISTRATION, DD214, ID_CARD, INSURANCE_CARD, IRS_SCHEDULE_B1_FORM_1065, IRS_SCHEDULE_C_OR_CEZ_PAGE_1, IRS_SCHEDULE_G_FORM_1120, IRS_SCHEDULE_K1_FORM_1065, IRS_SCHEDULE_K1_FORM_1120S, LEASE_AGREEMENT, LETTER_OF_AUTHORIZATION, OFFICIAL_LETTER, OTHER, PAY_STUB, POWER_OF_ATTORNEY, REGISTRATION_RECEIPT, REPORT_CARD, SECRETARY_OF_STATE_FILING, TRANSCRIPT, TUITION_RECEIPT, VENDOR_STATEMENT

Response Details

Expected Response

A list of Statement objects
