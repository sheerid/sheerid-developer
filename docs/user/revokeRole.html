Revoke Role

Remove a role from the specified user

Request Details

HTTP Method(s)

DELETE

Resource URI

https://services-sandbox.sheerid.com/rest/0.5/user/{userId}/role/{role} https://services.sheerid.com/rest/0.5/user/{userId}/role/{role}

Path Parameters

Name Type Description
userId string the user id of the user from whom to remove the role
role UserRole the role to be removed
Possible values: CUSTOMER_SERVICE, ORGANIZATION_MANAGER, PROMOTION_MANAGER, REPORTER, RESOURCE_MANAGER, REVIEWER, REVIEWER_MANAGER, REVIEW_REPORTER, REWARD_ADMIN, TEMPLATE_ADMIN, USER_ADMIN

Response Details

Expected Response

A single ResultWrapper object

Response Content Type(s)

application/json
