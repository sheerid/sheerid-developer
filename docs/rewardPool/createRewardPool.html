Create Reward Pool

Create a reward pool. A reward pool is a group of unique resources that are intended to be distributed to verified end-users. The pool is used by the reward service to issue a unique reward to each verified user. Examples of pooled rewards include single-use promotional codes or activation keys.

Request Details

HTTP Method(s)

POST

Resource URI

https://services-sandbox.sheerid.com/rest/0.5/rewardPool https://services.sheerid.com/rest/0.5/rewardPool

Parameters

Name Type Description
name string a reference name or description for this reward pool
warnThreshold int the number of entries at which the system will notify that a pool is running low on entries

Response Details

Expected Response

A single RewardPool object

Response Content Type(s)

application/json
