Update Reward Pool

Update a reward pool.

Request Details

HTTP Method(s)

PUT

Resource URI

https://services-sandbox.sheerid.com/rest/0.5/rewardPool/{rewardPoolId} https://services.sheerid.com/rest/0.5/rewardPool/{rewardPoolId}

Path Parameters

Name Type Description
rewardPoolId string the identifier of the reward pool to retrieve

Parameters

Name Type Description
name string a reference name or description for this reward pool
warnThreshold int the number of entries at which the system will notify that a pool is running low on entries

Response Details

Expected Response

A single RewardPool object

Response Content Type(s)

application/json
