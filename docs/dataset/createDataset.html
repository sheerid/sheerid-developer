Create Dataset

Create a new dataset to use for verification

Request Details

HTTP Method(s)

POST

Resource URI

https://services-sandbox.sheerid.com/rest/0.5/dataset https://services.sheerid.com/rest/0.5/dataset

Accepted Content Type(s)

application/x-www-form-urlencoded

Parameters

Name Type Description
affiliationType AffiliationType the type of affiliation represented by this dataset, will be added to the verification response upon successful match
Possible values: ACTIVE_DUTY, ALUMNUS, BUSINESS, BUSINESS_CONTACT, CIVILIAN_EMPLOYEE, CONTRACTOR, CUSTOMER, DISABLED_VETERAN, EMPLOYEE, EMT, FACULTY, FIREFIGHTER, GRADUATE, MEMBER, MILITARY_FAMILY, MILITARY_RETIREE, NON_PROFIT, NURSE, OWNER, POLICE, RESELLER, RESERVIST, STUDENT_COLLEGE_BOUND, STUDENT_FULL_TIME, STUDENT_PART_TIME, VETERAN
fields PersonFieldAccessor[] a list of fields that will be used to define a unique dataset entry
matchName boolean false to disable matching of organizationName. If organizationId is supplied or organizationName is not supplied, this parameter is ignored. (default value: true)
name string a reference name for this dataset
orgParameters OrganizationParameter[]
organizationId long an optional organization id
organizationName string an organization name supplied when the ID is not known. If organizationId is supplied, this parameter is ignored.
organizationType OrganizationType an organization type (see List Organization Types) to help when matching the organization name. If organizationId is supplied or organizationName is not supplied, this parameter is ignored.
Possible values: CORPORATE, FIRST_RESPONDER, K12, MEDICAL, MEMBERSHIP, MILITARY, NON_PROFIT, UNIVERSITY

Response Details

Expected Response

A single Dataset object

Response Content Type(s)

application/json
